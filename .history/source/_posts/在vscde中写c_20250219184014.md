# 1.下载mingw-w64
在网盘里了
将bin的路径添加到path中
# 2. VS从的中下载c/c++插件
ctrl+shift+p -》搜索c/c++，选ui配置-》下面有个模式也要改成mingw64
# 3. 配置
tasks.json中修改为这个
```

				"-g",
				"${workspaceFolder}\\*.c",
				"-o",
				"${workspaceFolder}\\${workspaceRootFolderName}.exe"


```
